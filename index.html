📖 Grammar — Stem-Changing Verbs (e→ie, e→i, o→ue) ▾

The "Boot" Pattern — stem changes in ALL forms EXCEPT nosotros

e→ie examplesquerer, entender, preferir, empezar, pensar, sentir
yo/tú/él/ellosquiero / quieres / quiere / quieren
nosotrosqueremos (NO change)
e→i examplespedir, servir, seguir, vestirse, repetir
yo/tú/él/ellospido / pides / pide / piden
o→ue examplespoder, dormir, volver, acostarse, almorzar
yo/tú/él/ellospuedo / puedes / puede / pueden

⚠ Venir and tener are e→ie BUT also have irregular yo: vengo, tengo.

Practice Quiz · Stem-Changers

📖 Grammar — The Imperfect Tense ▾

Regular Imperfect Endings

-AR (hablar)-ER/-IR (comer/vivir)
yohablabacomía / vivía
túhablabascomías / vivías
él/ellahablabacomía / vivía
nosotroshablábamoscomíamos / vivíamos
ellos/Uds.hablabancomían / vivían

⚠ Only 3 IRREGULAR verbs in the imperfect:

SERera / eras / era / éramos / eran
IRiba / ibas / iba / íbamos / iban
VERveía / veías / veía / veíamos / veían

KEY USES of the Imperfect

Habitual/repeatedCuando era niño, siempre jugaba en el parque.
Ongoing stateMi abuela tenía el pelo largo y hablaba tres idiomas.
Background/sceneLlovía y hacía frío cuando llegamos.
Time/age in pastEra las 3 de la tarde. Tenía 10 años.

Trigger words: siempre, todos los días, de niño/a, antes, generalmente, cada año

📖 Grammar — Preterite vs. Imperfect · Ya no / Todavía ▾

Preterite vs. Imperfect — "Movie Scene" Analogy

Preterite = plot eventsDe repente, empezó a llover. / Se casó en 1998.
Imperfect = backgroundHacía frío y el cielo estaba nublado.
TogetherLlovía (IMP) cuando llegamos (PRET) a la reunión.
Ya no + presentAntes vivía con mis padres. Ya no vivo con ellos.
Todavía + presentMi abuelo todavía trabaja aunque tiene 75 años.

📖 Grammar — Object Pronouns (DOP + IOP) ▾

Direct Object Pronouns (DOPs) — answers WHAT? or WHOM?

metelo / lanoslos / las

¿Ves la película? → Sí, la veo. / Quiero leerlo esta noche.

Indirect Object Pronouns (IOPs) — answers TO/FOR WHOM?

metelenosles

Le presté mi libro a Elena. / ¿Me puedes explicar la gramática?

Double Object Pronouns — IOP always comes FIRST

Rulele/les + lo/la/los/las → se + lo/la/los/las
ExampleLe di el libro → Se lo di.
ExampleLes mandó las fotos → Se las mandó.
Practice Quiz · Object Pronouns
